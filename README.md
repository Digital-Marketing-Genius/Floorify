# Floorify Canada

The source for the redesigned Floorify Canada website.

## Local development

Requirements: Node.js 22.13 or newer.

```bash
npm install
npm run dev
```

Create a production build with:

```bash
npm run build
```

## Main website sections

- Flooring and window-fashion product pages
- Flooring services and Calgary-area service pages
- Interactive virtual showroom
- Residential and commercial project portfolio
- Migrated Floorify flooring blog
- Estimate request and WhatsApp contact paths

The application is built with React, Next-compatible Vinext routing, and the
existing Floorify design system.
